# Feature Extractor
FastextreID是一个研究平台，实现了最先进的特征提取和再识别算法。Feature Extractor项目是**FastextreID**的一部分，为计算机视觉领域的开发人员提供了一个简单的工具包，可以从图像中提取特征。
